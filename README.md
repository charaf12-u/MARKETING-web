# 🚀 MarketPro – Newsletter & Weekly Marketing Plan

A premium, responsive, and animated marketing landing page designed to capture leads and organize content strategies.

## ✨ Features

- **Professional Design**: Modern UI with a dark theme, utilizing the sleek 'Inter' font family.
- **Interactive Elements**:
    - **Custom Flight Animation**: A dynamic plane animation that triggers on load.
    - **Spinning Badge**: A rotating call-to-action badge.
    - **Fading FX**: Smooth transition effects for decorative elements.
- **Weekly Strategy Grid**: A dedicated, visually appealing section for a 7-day marketing plan.
- **Responsive Layout**: Optimized for various screen sizes using modern CSS Flexbox and Grid.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with advanced animations, custom properties, and grid systems.
- **Google Fonts**: Inter.
